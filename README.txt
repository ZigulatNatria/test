В задании использованна дефолтная база для Django SQLite.

Для запуска проекта необходимо установить:
Django (применялась версия 4.1.2)
djangorestframework (применялась версия 3.14.0)

после установки перейти в каталог, содержащий файл manage.py
и запустить проект командой python manage.py runserver
перейти по ссылке: Starting development server at http://127.0.0.1:8000/
использовать в браузере следующие пути:
http://127.0.0.1:8000/v1/auth/register
http://127.0.0.1:8000/v1/auth/login
http://127.0.0.1:8000/v1/user (с query-параметром id)
